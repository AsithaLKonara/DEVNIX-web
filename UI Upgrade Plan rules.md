# 🎨 VISUAL SYSTEM (REFINED)

## 🌌 Background

* Global looping **dark cinematic video** (very subtle, low brightness)
* Overlay: gradient mask (dark → sapphire tones)
* Add **grain/noise layer** for premium feel

---

## 🎨 Color System (refined)

Your palette (adjusted for usability):

* Primary: `#7BA4D0` (soft sapphire blue)
* Deep: `#0D2440` (main background overlay)
* Accent: `#2E5E99` (hover, highlights)
* Light text: `#E7F0FA` (main text)

👉 Add:

* Glow accent: `rgba(123,164,208,0.25)`
* Border glass: `rgba(255,255,255,0.08)`

---

## 🧊 Glass System

* Backdrop blur: 12px–20px
* Border: 1px soft white
* Inner shadow subtle
* Slight reflection gradient

👉 Don’t overuse—use only on:

* cards
* nav
* CTA panels

---

## 🔤 Typography Strategy

Use max **2–3 fonts only** (very important)

Recommended combo:

* Headings: **GEONOVG or GORODA**
* Body: **Inter**
* Accent (rare use): **BREGAN or ALTRON**

👉 Too many fonts = kills premium feel
